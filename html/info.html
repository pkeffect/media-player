Internode

A professional-grade, browser-based media workstation featuring high-fidelity playback, precision audio shaping, video analysis tools, and persistent library management. Built entirely on modern Web Standards with full mobile and cross-browser support.

◈ Audio Engineering

12-Band Stereo Parametric EQ Professional frequency shaping from 25Hz to 16kHz with independent Left/Right channel control using Web Audio API Biquad Filters.
Stereo Width Control Mid/Side matrix processing for adjusting stereo field width from mono collapse to enhanced stereo spread.
Pre-Amp Input Gain Per-channel input gain knobs (±12dB) to boost or attenuate signals before EQ processing.
Safety Limiter Brickwall limiter prevents clipping and protects speakers from sudden volume spikes.
Real-Time VU Metering Live stereo RMS level visualization for monitoring output levels.
12 Genre Presets Calibrated EQ curves: Flat, Rock, Pop, Jazz, Classical, Techno, Dance, Vocal, Live, Full Bass, Full Treble, and Soft.
EQ Bypass Instantly defeat all processing for A/B comparison against the original source.
Crossfade Transitions Smooth audio fade-in/fade-out on play, pause, and track changes for seamless listening.

◈ Video Analysis & Playback

Adjustment Filters Non-destructive CSS filters for Brightness, Contrast, Saturation, and Hue. Essential for correcting dark or washed-out video streams.
A-B Loop Mark "Start" and "End" points on the timeline to loop specific segments for detailed analysis or learning.
Aspect Ratio Override Force video to 16:9, 4:3, 21:9, 2.35:1, 1:1, or Fill to correct videos with bad metadata.
Frame-by-Frame Advance Step through video one frame at a time (forward or backward) for precise navigation.
HLS Live Streaming Native Safari HLS support plus hls.js fallback for Chrome/Firefox. Adaptive bitrate with automatic quality switching.
Variable Speed Pitch-corrected playback from 0.25× to 4.0× for analysis, learning, or rapid consumption.
Picture-in-Picture Detach video to a floating window for system-wide multitasking.
Screenshot Capture Save the current video frame as a PNG image with one click.

◈ Advanced Subtitles

Format Support Native support for .SRT, .VTT, and fully styled .ASS / .SSA anime subtitles via the integrated Octopus engine.
Customizable Appearance Adjust font size, text color, and background opacity via the Caption Settings panel (right-click subtitle button).
Persistence Subtitle visibility state is saved and restored automatically across sessions.
Security Text rendering uses secure methods to prevent XSS attacks in malicious subtitle files.

◈ Library & Performance

Watched Folders Set a local library folder that automatically loads on startup using the File System Access API.
Diff-Based Rendering High-performance playlist engine updates only changed elements, allowing libraries of 1000+ items without UI freezing.
IndexedDB Persistence Stores playlists and metadata in the browser's structured database with debounced saves for reliability.
Thumbnail Batching Background generation of video thumbnails uses a constrained queue to prevent browser lock-up.
Smart Resume Automatically saves and restores playback position for every track across browser sessions.
Stream Validation Network streams undergo protocol checks (HTTP/HTTPS) and connectivity testing (HEAD request) before addition.

◈ Mobile & Interaction

Custom Keybindings Complete keyboard remapping support via the Help menu. Settings are persistent.
Touch Gestures Double-tap edges to seek, center to toggle UI. Swipe seeking on progress bar.
Playlist Management Drag and drop reordering, context menu actions (Move Up/Down, Remove), and sorting by Name/Date/Size.
Auto-Hide Controls UI fades away during playback for distraction-free viewing.
